# dotfiles

### Install

Includes:

* [Xcode Command Line Tools][xclt]
* [Homebrew][hb]
* [Oh My Zsh][omz]

```sh
$ git clone https://github.com/abendy/dotfiles.git ~/.dotfiles && cd ~/.dotfiles
$ sh bootstrap
$ cp localrc ~/.localrc
$ vi ~/.localrc
$ sh .osx
```

   [xclt]: <https://developer.apple.com/downloads>
   [hb]: <http://brew.sh>
   [omz]: <https://github.com/robbyrussell/oh-my-zsh>
