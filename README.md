# dotfiles

### Install

Also installs:

* [Xcode Command Line Tools][xclt]
* [Homebrew][hb]

```sh
$ git clone git@github.com:abendy/dotfiles.git ~/.dotfiles
$ cd dotfiles
$ ./bootstrap
$ sh .osx
$ sh Caskfile
$ brew tap homebrew/bundle && brew bundle Brewfile
```

   [xclt]: <https://developer.apple.com/downloads>
   [hb]: <http://brew.sh>
